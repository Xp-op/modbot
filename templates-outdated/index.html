Start using ModBot - 99% UPTIME!

ModBot is a multi-purpose moderation bot with a lot of commands and a swear word filter.
It's simple to use and it hasn't any configuration: just invite him and the moderation will start work.
ModBot has a swear wolds filter, and anyone who chat any forbidden word will be warned in DM. If an user says more than 3 bad words he will be banned.
You can change ModBot's prefix for your server by typing "/setprefix new_prefix" where new_prefix is the prefix you want. If you forgot your prefix, simply ping (mention) ModBot. You can change the prefix as many time as you want.
