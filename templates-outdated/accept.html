Accept configuration

To use the /accept command follow these instructions or start from this server model.

  • Go to your server's settings and create a "Verified" role. Do not edit any setting for this role.
  • Make all channels unavailable to the everyone role and available to the verified role: go to the @everyone role's settings and disable the "Read text channels & See voice channels" flag.
  • Create a bot role and assing the full admin permissions to this role. Assign this role to ModBot
  • Make sure that the verified role can view the server's channels
  • All done! Everything should work correctly, if not join my support server.
