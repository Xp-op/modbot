ModBot commands
We provide a detailed list of commands to make more simple to use our bot.
/addword PRO

Add your own custom word to your server's blacklist! Just type "/addword banned" changing banned with the word that you want! Note that you must muy premium before use this command! To get more info about this command, type "/help pro".





/rmword PRO

Remove your own custom word from your server's blacklist! Just type "/rmword banned" changing banned with the word that you want to remove! Note that you must muy premium before use this command and you can only remove the words that you have banned with the /addword command! To get more info about this command, type "/help pro".

/ban

This is a simple command that let you ban any user in the server. For use this, just type "/ban @user_mention" where @user_mention is a mention of the user to ban. Only users who have permission to ban members can use this command.





/kick

This is another command that let you kick any user in the server. For use this, just type "/kick @user_mention" where @user_mention is a mention of the user to kick. Only users who have permission to kick members can use this command.





/unban

This command let you unban a banned user. For use this command, chat "/unban user_name#1234" where user_name is the user to unban and 1234 is the member discriminator. Only users who have permission to manage the server can use this command.





/info

Use this command if you want to see detailed info of a member, like the day and time he/she joined your server and the number of roles he/she has. Just type "/info @user_mention", where @user_mention is a ping to the user you want to get info. All the members of your server can use this command.




/clean

With this command you can delete as many messages as you want, just chat "/clean 1234" where 1234 is the amount of messages to delete. Only users who has permission to manage message can use this command.



/accept

With this command you can accept the server rules. For the full configuration instructions click here.








/mute

Use this commands to mute an user. To get this command work, you need a "Muted" role without the send messages permission. For use this just chat "/mute @user_mention". You can't mute/unmute admins/moderators. This command only works for unmuted users.

/unmute

Use this commands to unmute an user. To get this command work, you need a "Muted" role without the send messages permission. For use this just chat "/unmute @user_mention". You can't mute/unmute admins/moderators. This command only works for muted users.

/new

Use this commands to create a text channel. Just type "/new name of the text channel". ModBot will automatically replace spaces with "-". You can also use /mkdir and /add.





/rm

Use this commands to delete a text or a vocal channel. Just type "/rm name-of-the-text-channel" or "/rm Name of the vocal Channel". Remember to replace spaces with "-". You can also use /remove and /delete.



/setprefix

Costumize your server's prefix. To use this command you must be an admin and you have to type "/setprefix new_prefix" where new_prefix is the prefix you want. If you forgot your prefix, ping (mention) ModBot and he'll send it to you. You can change the prefix unlimited times.

/invite

Create an instant invite to your server. Note that the generated invite will never expire and it has unlimited uses.








/set-swear

Enable or disable the swear words filter. Use this by typing "/set-swear on/off". Off is for disable the filter, on is for enable it. Type /help in the chat for more info.






/whitelist

Whitelist a member, just type "/whitelist @user_mention" where @user_mention is a mention to the user to whitelist.







/blacklist

Blacklist a member, just type "/blacklist @user_mention" where @user_mention is a mention to the user to blacklist.







/help

Get help for all my commands. Type "/help pro" to get info about premium
